PlayM8

Your next favorite game is just a swipe away.

PlayM8 is an app for all. Whether you're looking to find your next obsession, or you are entirely new to video games, we create an interactive experience that replaces tedious searching with an intuitive swiping mechanic.

App Features

Quizzing

New to gaming? That's okay! Tell us what consoles you own and what genres you like, and we'll build a custom profile to get you started.

Swiping

Swipe through tons of games until you find the one(s) just for you. Building your library has never been this interactive.

Linking

Have preexisting gaming accounts? Link your Steam account directly to PlayM8 to make it even easier to find that perfect game based on your history.

The Tech Stack

Frontend

Cross-platform mobile application built entirely in Flutter/Dart for a seamless swiping experience on any device.

Backend & Database

REST API operations written in Python, connected to a robust PostgreSQL database with built-in user authentication.

Live Integrations

Pulling real-time, accurate game data and cover art utilizing the IGDB API and the Steam Web API.
